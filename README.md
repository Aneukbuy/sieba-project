# SIETBA - Sistem Informasi Event Terbuka Banda Aceh

## Deskripsi Proyek

SIETBA (Sistem Informasi Event Terbuka Banda Aceh) adalah platform web modern yang menyediakan informasi lengkap tentang berbagai event terbuka yang diselenggarakan di Banda Aceh. Website ini dirancang dengan antarmuka yang responsif dan user-friendly untuk memudahkan masyarakat dalam menemukan dan mengikuti event-event menarik di kota Serambi Mekkah.

## Fitur Utama

### 🎯 **Fitur Utama**
- **Informasi Event Terlengkap**: Database event dengan kategori Budaya, Pendidikan, Olahraga, dan Ekonomi
- **Filtering & Pencarian**: Filter event berdasarkan kategori dan pencarian berdasarkan judul, deskripsi, atau lokasi
- **Responsive Design**: Tampilan optimal di desktop, tablet, dan mobile
- **Interactive UI**: Animasi smooth dan efek hover yang menarik
- **Contact Form**: Formulir kontak untuk pertanyaan dan saran
- **Newsletter**: Sistem berlangganan newsletter untuk update event terbaru

### 🎨 **Desain & UX**
- **Modern UI/UX**: Desain clean dan modern dengan gradien warna yang menarik
- **Mobile-First**: Responsif di semua perangkat
- **Smooth Animations**: Animasi fade-in dan hover effects
- **Accessibility**: Navigasi yang mudah dan aksesibel

### 📱 **Responsivitas**
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (320px - 767px)

## Teknologi yang Digunakan

- **HTML5**: Struktur semantic dan modern
- **CSS3**: 
  - Flexbox & Grid untuk layout
  - CSS Variables untuk konsistensi warna
  - Media Queries untuk responsivitas
  - Animations & Transitions
- **JavaScript (ES6+)**:
  - DOM Manipulation
  - Event Handling
  - Filtering & Search
  - Form Validation
  - Intersection Observer API
